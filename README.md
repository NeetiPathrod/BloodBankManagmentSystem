# Blood Bank Management System

This project is created as a college assignment.

## Backend
- Java (Console Based Application)
- Features:
  - Add Donor
  - View Donor
  - Exit

## Frontend
- HTML & CSS
- Red and White theme
- Demo UI for Blood Bank System

## How to Run Backend
1. Open the project in NetBeans
2. Run `Main.java`
3. Use console to enter choices

## Frontend Demo
1. Open `index.html` in any browser
2. Fill the form (demo purpose only)

## Note
Frontend is for demonstration.
Actual logic runs in Java console.
